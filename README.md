# Webcam Image Capture

This Python script captures a frame from the webcam using OpenCV and saves it as 'test.jpg'.

## Prerequisites

- Python 3.x
- OpenCV

## Usage

1. Install the required dependencies:

   ```bash
   pip install opencv-python
   ```

2. Run the script:

   ```bash
   python capture_image.py
   ```
