# Webcam Image Capture with OpenCV

This Python script captures a frame from the webcam using OpenCV and saves it as 'test.jpg'.

## Prerequisites

- Python 3.x
- OpenCV (cv2)

## Installation

Install the required dependencies using pip:

```bash
pip install opencv-python
```

## Usage

Run the script to capture an image from the webcam:

```bash
python capture_image.py
```

The captured image will be saved as 'test.jpg' in the project directory.
